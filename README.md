_JTC_Oak:_ Open academic network visualization
=======================

_JTC_Oak: Open Academic Network_ aims to facilitate dialogue amongst people working on social, ecological, and policy ramifications of regions undergoing structural change. We have created a social network visualization for this purpose.This visualization maps scholars belonging to the _JTC_Oak: Open Academic Network_ at the Just Transition Center, Martin-Luther-Universität Halle-Wittenberg. It is created for information purposes only, and in the spirit of creating an open exchange between scholars working on just transition topics.

The network visualization was created by <a href="https://jtc.uni-halle.de/en/dr-gregory-gan/">Dr. Gregory Gan</a> with assistance from Susanna Seguro Rütz and Milan Harms who helped assembled the data contained therein, and based on Vasco Asturiano's <a href="https://vasturiano.github.io/3d-force-graph/">3d-force-graph</a>.

Please see the full visualization <a href="https://greg-gan.github.io/JTC_Oak-visualization/public/">here</a>.
