_JTC_Oak:_ Open academic network visualization
=======================
This social network visualization maps scholars belonging to the _JTC_Oak: Open Academic Network_ at the Just Transition Center, Martin-Luther-Universität Halle-Wittenberg. It is created for information purposes only, and in the spirit of creating an open exchange between scholars working on just transition topics

The visualization was created by <a href="https://jtc.uni-halle.de/en/dr-gregory-gan/">Dr. Gregory Gan</a> and based on Vasco Asturiano's <a href="https://vasturiano.github.io/3d-force-graph/">3d-force-graph</a>.

Please see the visualization <a href="https://greg-gan.github.io/JTC_Oak_visualization/public">here</a>.
